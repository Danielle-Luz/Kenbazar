Confira e utilize o projeto no <a href="https://danielle-luz.github.io/Kenbazar/">github pages</a>.


<h2 id="descricao">:pencil2: Descrição</h2>
O Kenbazar é uma e-commerce fictícia de eletrônicos que utiliza os conceitos de DOM para renderização dinâmica e o local storage para persistência dos produtos adicionados ao carrinho.


<h2 id="funcionalidades">:hammer: Funcionalidades</h2>
<div align=center>
    <img src="src/img/readme/kenbazar.gif">
</div>

- Responsividade.
- Filtragem por categorias.
- Pesquisa dinâmica conforme os dados são digitados na barra de busca.
- Adicionar, remover e aumentar a quantidade de itens no carrinho.
